## 재학생 리그

### 행사 일정
* 12월 9일 ~ 12월 27일 : 참가자 모집
* 12월 9일 ~ 1월 4일 : 온라인 자율학습형 교육(분야 : AI, 클라우드 / 참가신청 즉시 교육시작 가능)
* 1월 4일 : 오프라인 교육(스터디 윗미, QnA) - 연암공대 강의실 예정
* 1월 10일 : 예선 결과물 제출
* 1월 11일 ~ 1월 13일 : 예선 심사
* 1월 13일 : 본선 진출자 통보
* 1월 14일 : 본선 심사 및 시상

### 참가 대상
- 연암공과대학교 재학생

### 주제
- #### 연암공대의 다양한 문제를 해결할 수 있는 서비스 및 웹/앱 개발 : 팀워크(가점사항)
  * (예시 1) 연암공대 내 유휴공간을 실시간으로 확인하고 시간별로 빌릴 수 있는 어플
  * (예시 2) 교내 여러 학생들과 소통하고 팀원을 구할 수 있는 웹
 
### 시상
- (추후 업데이트 예정)
